# Vaypr Project Instructions

## Structure
- Frontend lives in frontend/
- Backend lives in backend/
- Deployment uses Docker / docker-compose
- MongoDB is hosted on MongoDB Atlas
- Email is handled through Brevo
- Production runs on VPS

## Rules
- Never commit .env files or real secrets
- Use .env.example for required environment variables
- Test frontend and backend before deployment
- Make changes on branches, not directly on main
